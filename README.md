# Sudoku
Feito em Vanilla JS usando o bundler Vite e gerenciador de pacotes Bun

# Como rodar o projeto

Primeiro baixe esse repositório
```bash
git clone git@github.com:jvnascimento2007/sudoku.git
cd sudoku
```

Instale dependências e rode o servidor de desenvolvimento
```bash
bun install
bun run dev
```

Faça build e veja o preview do servidor de produção
```bash
bun run build
bun run preview
```
